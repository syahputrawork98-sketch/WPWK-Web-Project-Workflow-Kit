# Docs

Folder ini berisi seluruh dokumentasi project, baik yang terkait dengan sistem workflow, riwayat pengembangan, maupun detail teknis per area (frontend, backend, database, deployment).

- `docs/project/`: Digunakan sebagai workflow/control layer (onboarding, prompt, aturan kerja).
- `docs/history/`: Digunakan sebagai persistent project memory layer (status project, feature tracking).
- `docs/frontend/`, `docs/backend/`, `docs/database/`, `docs/deployment/`: Digunakan untuk dokumentasi teknis tiap area.

Gunakan folder ini sebagai panduan utama selama siklus pengembangan berlangsung.

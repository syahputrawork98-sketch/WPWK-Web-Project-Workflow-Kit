# Specialist Analysis Template — WPWK

Fungsi:
Template ini digunakan oleh roomchat spesialis saat user membutuhkan analisa mendalam sebelum dikirim ke Roomchat 00.

Format wajib:

Specialist Analysis Summary

Specialist Room:
[UX / Frontend / Backend / Database / Auth / Deployment / Content / Business / Feature Planning / lainnya]

Topic:
[Topik yang dianalisis]

Purpose:
[Tujuan analisa]

Context:
[Konteks project atau fitur]

Recommendation:
[Rekomendasi utama]

Options Considered:
1. [Opsi 1]
2. [Opsi 2]
3. [Opsi 3]

Risk Notes:
* [Risiko 1]
* [Risiko 2]

Suggested Scope:
* [Scope yang disarankan]

Suggested Next Batch:
Batch FXXA — [Nama batch yang disarankan]

Not Recommended:
* [Hal yang sebaiknya tidak dilakukan]

Decision Needed from Roomchat 00:
* [Keputusan yang harus dibuat Roomchat 00]

Aturan:
* Roomchat spesialis tidak memberi instruksi final langsung ke Gemini.
* Roomchat spesialis tidak menentukan batch final.
* Roomchat spesialis tidak mengubah status project.
* Roomchat spesialis tidak otomatis menjadi dokumen repo.
* Ringkasan spesialis harus dikirim ke Roomchat 00.
* Roomchat 00 yang menerima, menolak, menyederhanakan, atau mengubah hasil analisa menjadi batch eksekusi.

# Welcome to WPWK (Web Project Workflow Kit)

WPWK adalah template workflow untuk project web yang mengintegrasikan AI (ChatGPT, Gemini) dengan arsitektur yang sangat terkontrol.

Repositori ini kini mendukung dua jalur utama (Mode). Silakan pilih mode mana yang sesuai dengan kondisi project Anda saat ini.

## 1. Greenfield Mode (Project Belum Ada)
Gunakan mode ini jika Anda ingin membangun aplikasi dari nol. Folder `WK-Web-Greenfield/` telah disiapkan dengan struktur boilerplate dasar (`client/` dan `server/`). AI akan membantu Anda membangun fondasi dan arsitektur dari tahap paling awal.

👉 **Langkah Selanjutnya**: Baca panduan di `docs/project/workflow/GREENFIELD_MODE.md`

## 2. Existing Project Mode (Project Sudah Ada)
Gunakan mode ini jika project Anda sudah memiliki kode (`client/`, `server/`, atau struktur lainnya) dan Anda ingin memasukkan sistem dokumentasi & workflow WPWK ke dalamnya untuk mengontrol pengembangan selanjutnya.
Mode ini sangat mengutamakan kehati-hatian, dimulai dari proses Audit dan Pemetaan (Mapping) tanpa mengubah kode existing di awal.

👉 **Langkah Selanjutnya**: Baca panduan di `docs/project/workflow/EXISTING_MODE.md`

---
> **PENTING**: Pastikan Anda membaca dan memahami `docs/project/workflow/WORKING_SYSTEM.md` sebagai panduan dasar pengoperasian sistem ini, terlepas dari mode mana yang Anda pilih.
